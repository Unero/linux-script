## Know your shell
1. Determine your shell first:
  1. In Terminal ``` echo $0 ```  
1. To change your shell:
  1. Check shell list installed in your system:
    1. In Terminal ``` cat /etc/shells ```
1. In the first line of script add ``` # !shell-path ```
